# 🌐 AI Screen Assistant - Landing Page

A premium, modern landing page designed to showcase the **AI Screen Assistant** project. This website provides a visual overview of the system's vision and voice capabilities, features, and underlying tech stack.

## ✨ Features

- **Hero Showcase:** High-impact "Vision Meets Voice" introduction.
- **Terminal Simulation:** A live-style mockup demonstrating the assistant's logic in real-time.
- **Feature Grid:** Detailed breakdown of Global Hotkey, Instant Vision, Voice Interaction, and Safety Guard.
- **Tech Stack Overview:** Visual cards for Qwen2-VL, Whisper, PyTorch, and CustomTkinter.
- **Modern UI/UX:** Built with React, featuring glassmorphism, smooth animations, and a responsive dark-themed design.

## 🛠️ Tech Stack

- **Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling:** Vanilla CSS (Custom Glassmorphic Design System)
- **Typography:** Outfit & Inter (Google Fonts)

## 🚀 Getting Started

### Development
1. **Navigate to the website directory:**
   ```bash
   cd website
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```

### Production
To build the project for production:
```bash
npm run build
```
The output will be located in the `dist/` directory.
